1 — Create the wasteless roles in your AWS account

One CloudFormation stack creates wasteless-readonly — Describe/Get/List permissions only. This is the only part that happens inside your AWS account, and the button below pre-fills it all.

Opens your AWS console on a pre-filled Create stack review page: sign in as an administrator, tick "I acknowledge that AWS CloudFormation might create IAM resources with custom names", click Create stack, wait for CREATE_COMPLETE (~1 min), then come back here and hit Test connection — the fields in step 2 are already filled in. If someone else manages AWS, send them the link with Copy link. Type an External ID in step 2 before opening the link to bake it in.

Manual alternatives (no pre-filled link)
  • CloudFormation (no tooling needed): upload onboarding/cloudformation/wasteless-onboarding.yaml in the AWS console → Create stack
  • Terraform (only if your team already uses it): apply onboarding/terraform/

Full walkthrough: docs/CTO_QUICKSTART.md (short) or docs/AWS_SETUP.md (detailed, per-permission).
