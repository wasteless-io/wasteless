Connect your AWS account — about 10 minutes, read-only by default. Wasteless cannot modify anything unless you later create the optional remediation role.

Step 1

Create the wasteless roles in your AWS account

One CloudFormation stack, read-only (Describe/Get/List). The button pre-fills everything.

In the AWS console that opens:

  1. Sign in as an administrator.
  2. Tick the IAM acknowledgement, then Create stack.
  3. Wait ~1 min for CREATE_COMPLETE, then return to Step 2.
More options & help
  • Someone else manages AWS? Send them the Copy link.
  • Using an External ID? Enter it in Step 2 before opening the link.
  • No pre-filled link: upload onboarding/cloudformation/wasteless-onboarding.yaml in the AWS console, or apply onboarding/terraform/.
  • Full walkthrough: docs/CTO_QUICKSTART.md or docs/AWS_SETUP.md.
